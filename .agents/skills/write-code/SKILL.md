# Backend Code по конвенциям

# Prerequisite

Инструкции о том, что писать, должны быть предоставлены — либо напрямую пользователем,
либо иным образом.

## Шаг 1: Загрузи конвенции

Загрузи конвенции из Dev Book и применяй их.

## Шаг 2: Пиши код по конвенциям

Следуй правилам из загруженных конвенций MCP. Не упоминай что загрузил конвенции — просто действуй по правилам.

## Шаг 3: Проверь TODO

Перед финальным кодом убедись что все новые TODO содержат ссылку на задачу в трекере:

```
// TODO: BIGD-XXXX описание
```

Если TODO без ссылки — предупреди явно и не двигайся дальше пока не будет указан номер задачи.
